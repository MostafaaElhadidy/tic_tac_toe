# Tic Tac Toe (Python OOP)

A simple **command-line Tic Tac Toe game** implemented in Python using **Object-Oriented Programming (OOP)** principles.  
Players can play multiple rounds, and the game tracks turns, wins, ties, and allows replaying.

---

## Features

- 2-player game (Human vs Human)
- Board displayed in the terminal
- Checks for **win**, **tie**, and **invalid moves**
- Option to **play again** after each round
- Fully implemented with **OOP** (Board, Player, Game classes)

---

## File Structure

tic_tac_toe/
- ├── board.py # Board class (grid, print, update, win/tie check)
- ├── player.py # Player class (symbol, move input)
- ├── game.py # Game class (main game loop, turn handling)
- └── main.py # Entry point to start the game
